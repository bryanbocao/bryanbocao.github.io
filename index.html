Research

I work on multimodal learning (vision + IMU + Wi-Fi FTM), efficient machine learning (few-class evaluation, pruning, model selection), and robot visual navigation (hierarchical RL, metric learning, 3D Gaussian Splatting).

Current focus

  • Vision–wireless identity association & cross-modal trajectory reconstruction
  • Few-class regime: benchmark tooling + dataset difficulty measurement
  • Efficient inference: pruning and right-sized model selection
  • Embodied navigation: representation learning and mapping
